Getting Started with SQL

SQL stands for Structured Query Language
With SQL we can store data in a table, similar to a spreadsheet with rows and columns.
For Example:
CREATE TABLE exercise_logs
(id INTEGER PRIMARY KEY AUTOINCREMENT, type TEXT, minutes INTEGER, calories INTEGER, heart_rate INTEGER);

INSERT INTO exercise_logs(type, minutes, calories, heart_rate) VALUES ("biking", 30, 100, 110);
INSERT INTO exercise_logs(type, minutes, calories, heart_rate) VALUES ("biking", 10, 30, 105);
INSERT INTO exercise_logs(type, minutes, calories, heart_rate) VALUES ("dancing", 15, 200, 120);
SELECT * FROM exercise_logs;
This table displays a graph pertinent to health and fitness and will appear as the following:

id type minutes calories heart_rate
1 biking 30 100 110
2 rock-climbin 45 500 95
3 dancing 20 200 120
